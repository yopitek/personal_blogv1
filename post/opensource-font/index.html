為什麼要用開源字體?

你可能聽過「微軟正黑體可以商用嗎?」這類問題。事實上,許多看似「免費」的字體,實際上只能個人使用,一旦用於商業用途(包括學校對外發布的海報、營隊宣傳品等)就可能構成侵權。

開源字體則完全不同:

  • 明確授權:清楚標示可商用、可修改、可散布
  • 永久免費:不用擔心授權到期或突然收費
  • 跨平台支援:Windows、macOS、Linux 都能用
  • 持續更新:社群會不斷改進字體品質

根據 justfont 的說明,開源字體與免費字體是不同概念——有些免費字體僅供個人非商業使用,而開源字體則保障了使用者的權利。

開源字體授權快速入門

GitHub 上的繁體中文字體主要使用兩種授權:

🔵 SIL Open Font License (OFL) 1.1

最常見的字體開源授權,約 80% 的開源中文字體都採用這個授權。

你可以:

  • ✅ 免費用於個人或商業用途
  • ✅ 印在書籍、海報、產品包裝上
  • ✅ 嵌入網站、App、影片中
  • ✅ 修改字體並重新發布(需同樣採用 OFL)

你不可以:

  • ❌ 單獨販售字體檔案本身
  • ❌ 修改後使用原字體名稱

簡單說:用字體做出來的作品(簡報、海報、影片等)完全自由,但字體檔案本身不能單獨賣錢。

🔵 Apache License 2.0

較少見但更寬鬆的授權。

主要差異:

  • ✅ 可以單獨販售字體
  • ✅ 修改後不一定要開源
  • ⚠️ 散布字體檔案時需附上 LICENSE 檔案

實務建議:對於教師和學生來說,只要記得「可以安心商用」就夠了!用這些字體做簡報、報告、海報、影片都沒問題。

如何從 GitHub 下載字體

許多老師和學生第一次看到 GitHub 可能會覺得有點複雜,別擔心!下載字體其實很簡單,跟著以下步驟操作即可:

方法一:下載已編譯好的字體檔案(推薦)

這是最簡單的方法,適合所有人:

步驟 1:進入 GitHub 字體頁面

以「jf open 粉圓」為例,開啟 https://github.com/justfont/open-huninn-font

步驟 2:找到 Releases 區域

在頁面右側欄位,找到「Releases」(發行版本)區塊,點擊最新版本號碼(例如「v2.1」)

![GitHub Releases 位置示意](示意圖:右側欄位會顯示 Releases 標題和版本號)

步驟 3:下載字體壓縮檔

往下捲動到「Assets」(資源)區塊,會看到多個 .zip 或 .7z 壓縮檔:

  • 選擇檔案名稱包含 .ttf 或 .otf 的壓縮檔
  • 點擊檔案名稱即開始下載

步驟 4:解壓縮

下載完成後,使用 Windows 內建解壓縮功能或 7-Zip、WinRAR 等軟體解壓縮,就能看到字體檔案了!

方法二:直接點擊綠色「Code」按鈕

某些字體專案會直接提供字體檔案:

  1. 點擊綠色「Code」按鈕
  2. 選擇「Download ZIP」
  3. 解壓縮後,在 fonts/ 或 dist/ 資料夾中找字體檔案

💡 小技巧:如何選擇字體格式

解壓縮後可能會看到不同格式:

  • .ttf(TrueType Font):最通用,Windows 和 macOS 都支援,建議優先選擇
  • .otf(OpenType Font):較新的格式,支援更多進階功能,也很通用
  • .woff/.woff2:網頁用格式,一般電腦安裝不需要

結論:下載 .ttf 或 .otf 檔案就對了!

字體安裝教學

下載字體後,安裝非常簡單:

🖥️ Windows 10/11 安裝方式

方法一:雙擊安裝(最簡單)

  1. 在下載的字體檔案(.ttf 或 .otf)上按兩下
  2. 會彈出字體預覽視窗
  3. 點擊左上角「安裝」按鈕
  4. 等待幾秒鐘,完成!

方法二:拖曳安裝(批次安裝多個字體時更快)

  1. 按 Win + R 鍵,輸入 fonts 後按 Enter
  2. 會開啟「字型」資料夾
  3. 把下載的字體檔案直接拖曳進去
  4. 等待複製完成即可

🍎 macOS 安裝方式

方法一:字體簿安裝

  1. 在字體檔案上按兩下
  2. 會開啟「字體簿」(Font Book)應用程式
  3. 點擊「安裝字體」按鈕
  4. 完成!

方法二:手動複製

  1. 開啟 Finder
  2. 按 Cmd + Shift + G,輸入 /Library/Fonts
  3. 把字體檔案拖曳進去

✅ 安裝完成後怎麼使用?

  1. 重新啟動 PowerPoint、Word、Photoshop 等軟體
  2. 在字體選單中搜尋字體名稱
  3. 開始使用!

注意:部分軟體(如 Photoshop、Illustrator)需要完全關閉後重新開啟,才能讀取新安裝的字體。

依使用情境推薦字體

不同場合適合不同字體風格,這裡提供實用的分類推薦:

📄 適合正式文件/論文的字體

首選:

  1. 思源宋體 TC(Regular 或 Medium):傳統、正式、易讀,學術論文最佳選擇
  2. 源樣明體(Regular):比思源宋體更接近台灣傳統印刷體習慣
  3. 思源黑體 TC(Regular):現代感強,正式但不古板

使用情境:

  • 碩博士論文內文
  • 正式報告
  • 學術海報的內文說明

搭配技巧:

  • 標題用黑體(Bold 或 SemiBold)
  • 內文用明體(Regular)
  • 行距建議設定為 1.5 倍或 2 倍

📊 適合簡報/教學的字體

首選:

  1. jf open 粉圓(Regular):圓潤可愛,投影清晰,特別適合國小到高中的教學簡報
  2. 思源黑體 TC(Medium 或 Bold):專業大方,大學簡報或研討會適用
  3. 源泉圓體(Medium):比粉圓更專業一點,但仍保持親切感

使用情境:

  • 教學簡報
  • 學生課堂報告
  • 線上課程字卡

搭配技巧:

  • 簡報標題用 Bold 或 Heavy 字重,提升視覺衝擊
  • 內文用 Regular 或 Medium,確保遠距離可讀性
  • 避免使用太細的字重(如 Light、ExtraLight),投影時會看不清楚

🎨 適合創意設計/海報的字體

首選:

  1. 霞鶩文楷 TC:手寫感楷體,適合文青風、懷舊風設計
  2. 芫荽體:輕鬆活潑,適合社團活動、營隊海報
  3. 源雲明體:傳統印刷體韻味,適合復古設計、文化主題
  4. jf open 粉圓:年輕、親切,適合活動宣傳、社群貼圖

使用情境:

  • 社團海報
  • 營隊宣傳單
  • 畢業紀念冊
  • IG 貼文圖

搭配技巧:

  • 混用不同風格字體(例如標題用粉圓,說明文字用黑體)
  • 善用字重對比(粗細搭配)
  • 創意字體建議用在標題,內文仍選擇易讀的黑體或明體

📚 特殊需求推薦

需要注音符號支援

  • jf open 粉圓:完整支援注音符號、臺語、客語拼音
  • 思源黑體 TC:支援注音符號

需要大量罕見字

  • 思源黑體 TC:支援 65,535 字圖,涵蓋罕用字、異體字
  • 思源宋體 TC:同上

需要多種字重

  • 思源系列(黑體、宋體):各有 7 種字重,從 ExtraLight 到 Heavy
  • 源樣、源流、源雲系列:同樣提供多種字重選擇

常見問題 Q&A

Q1:這些字體真的可以商用嗎?

A:是的!本文介紹的所有字體都採用 OFL 1.1 或 Apache 2.0 授權,明確允許商業使用。你可以用它們製作:

  • 販售的書籍、雜誌
  • 商業簡報
  • 產品包裝
  • YouTube 影片(可營利)
  • 付費課程教材

唯一的限制是:不能單獨販售字體檔案本身。

Q2:需要標註字體來源嗎?

A:依授權規定:

  • 使用字體製作的作品(簡報、海報、影片等):不需要標註
  • 散布字體檔案本身:需要附上原始授權檔案(LICENSE)

簡單說,老師做簡報、學生做報告,都不需要特別註明「本簡報使用 XX 字體」。但如果你要把字體檔案分享給別人,記得一併附上 LICENSE 檔案。

Q3:可以修改這些字體嗎?

A:可以!OFL 授權允許修改字體。但需要注意:

  • 修改後的字體也必須採用 OFL 授權
  • 不能使用原字體名稱(需改名,例如「思源黑體」改為「我的黑體」)

對於大多數使用者(教師、學生)來說,直接使用原版字體就很夠用了。

Q4:字體安裝後找不到怎麼辦?

A:可能原因和解決方法:

  1. 軟體未重新啟動:完全關閉 PowerPoint/Word 等軟體後重新開啟
  2. 搜尋字體名稱不正確:嘗試搜尋字體的英文名稱(例如「Source Han Sans TC」而非「思源黑體」)
  3. 字體安裝失敗:重新安裝,確認檔案沒有損壞
  4. macOS 權限問題:嘗試安裝到使用者資料夾 ~/Library/Fonts 而非系統資料夾

Q5:GitHub 下載速度很慢怎麼辦?

A:GitHub 在台灣的連線有時會比較慢,可以試試:

  1. 使用瀏覽器的下載管理器(支援斷點續傳)
  2. 等待離峰時段(晚上 23:00 後或清晨)下載
  3. 部分字體有提供 CDN 連結或鏡像站(可在字體專案的 README 中找到)

Q6:手機/平板可以安裝這些字體嗎?

A:視系統而定:

  • iOS/iPadOS:需要透過專用 App(如「AnyFont」)或使用描述檔安裝,較為複雜
  • Android:部分手機支援字體更換(如 Samsung、小米),可參考各品牌說明
  • 建議:行動裝置使用系統內建字體,或使用支援自訂字體的 App(如 Canva、Phonto)
