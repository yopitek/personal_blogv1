2026 Mac 新手裝機指南:學長姐私藏的 25 款必裝軟體

從系統工具到開發環境,25 款讓 Mac 更好用的必裝軟體推薦。包含 Raycast、Rectangle、VS Code 等工具,還有 Homebrew 一鍵安裝腳本,讓你的 Mac 戰力滿點!

Dec 01, 2025
閱讀時間: 4 分鐘

還記得我第一次打開 MacBook 的時候,整個人是懵的。

滑鼠右鍵怎麼按？視窗怎麼放大？檔案總管在哪？用了十幾年 Windows,換到 Mac 就像突然搬到國外,什麼都要重新學。

最痛苦的是,我花了快一個月才摸索出「哦原來 Mac 可以裝這些軟體讓它更好用」。那種「早知道就好了」的感覺,真的很 frustrating。

所以這篇就是我的「早知道清單」。這 25 款軟體,是我用了三年 Mac 後真心覺得必裝的。裝完之後,你會發現 Mac 不只好看,是真的好用。

系統工具:讓 Mac 更順手 🛠️

從 Windows 轉過來,最不習慣的就是「找不到熟悉的功能」。這幾個工具能讓你快速找回熟悉感。

Raycast - 超強啟動器

Mac 內建的 Spotlight 很陽春對吧？Raycast 是加強版,而且強大到你會懷疑人生。

按一個快捷鍵,就能搜檔案、開 App、查單字、算數學、控制音樂播放、切換 Wi-Fi⋯⋯甚至可以直接叫 AI 幫你改文案。它還內建剪貼簿歷史、視窗管理、番茄鐘,根本把十幾個 App 塞進一個啟動器裡。

如果你只能裝一個效率工具,就裝 Raycast。它會改變你用 Mac 的方式。

什麼時候用: 任何需要快速操作的時候 - 開檔案、搜尋、執行指令

Rectangle - 視窗管理神器

Windows 能用 Win + 方向鍵把視窗貼邊,Mac 原生做不到這點真的很痛。Rectangle 就是來解決這問題的。

按個快捷鍵,視窗自動分左半、右半、全螢幕、四分之一螢幕。寫程式碼配文件、看影片配做筆記,多工超方便。

什麼時候用: 需要同時看多個視窗(寫程式、剪片、做簡報)

Clipy - 剪貼簿歷史記錄

你有沒有過「剛剛複製的東西被覆蓋掉」的崩潰經驗？Clipy 幫你記住所有複製過的內容,隨時叫出來用。

按 ⌘+Shift+V 就能看到完整的複製歷史。寫文章、整理資料的時候超實用。

什麼時候用: 需要頻繁複製貼上(寫文章、整理資料、填表格)

AppCleaner - 軟體移除工具

Mac 移除軟體不能只丟到垃圾桶,很多設定檔和快取會殘留在系統裡。AppCleaner 幫你完全移除,連相關檔案都一起清。

用法超簡單,把 App 拖進去就搞定。

什麼時候用: 移除不要的軟體,保持系統乾淨

The Unarchiver - 解壓縮工具

Mac 內建的解壓縮只能處理 .zip,碰到 .rar、.7z 就 GG 了。The Unarchiver 支援幾乎所有壓縮格式,而且完全免費。

裝了它,你就不用煩惱「這個檔案怎麼打不開」。

什麼時候用: 下載壓縮檔、處理各種格式

PopClip - 選取文字增強工具

選取文字後,PopClip 會跳出一排功能鈕:複製、翻譯、搜尋、改大小寫、發推特⋯⋯你想得到的操作它都有。

就像手機上長按會跳出選單,PopClip 讓 Mac 的文字操作變超直覺。

什麼時候用: 大量處理文字(寫作、翻譯、整理資料)

Snipaste - 截圖標註神器

Mac 內建的截圖功能陽春到不行。Snipaste 不只截圖,還能直接在圖上畫箭頭、加文字、打馬賽克,截完的圖還可以釘在桌面上當便利貼。

寫教學文件、回報 bug、做簡報的時候超好用。

什麼時候用: 需要截圖並標註(教學、報告、溝通)

生產力工具:筆記、文書、學習 📝

工作和學習離不開這些工具。裝好它們,你的生產力直接起飛。

Notion - 全能筆記工具

Notion 根本是數位工作者的瑞士刀。筆記、待辦清單、專案管理、資料庫、知識庫⋯⋯全包了。

我用 Notion 管理所有專案、寫文章大綱、追蹤學習進度。它的資料庫功能特別強,可以做出很彈性的資訊管理系統。

什麼時候用: 專案管理、團隊協作、個人知識庫

Obsidian - 本地筆記利器

如果你喜歡把筆記存在自己電腦,Obsidian 是最佳選擇。它用 Markdown 格式,筆記之間可以互相連結,慢慢建立起自己的知識網絡。

很多人用它做第二大腦、寫論文、整理讀書筆記。資料都在本機,不怕雲端服務掛掉。

什麼時候用: 長期知識累積、學術研究、注重隱私

Anki - 記憶卡片神器

要背單字、準備考試、記專業術語,Anki 是最強工具。它用「間隔重複」演算法,在你快忘記的時候提醒你複習,記憶效率超高。

醫學生、考證照的、學語言的都在用。介面雖然有點醜,但效果真的好。

什麼時候用: 考試準備、背單字、記大量資訊

LibreOffice - 免費文書處理

不想付 Microsoft Office 的訂閱費？LibreOffice 是開源免費的替代方案,該有的功能都有。

Word、Excel、PowerPoint 的檔案都能開,基本文書處理完全夠用。偶爾要編輯 Office 文件,裝這個就對了。

什麼時候用: 文書處理、偶爾要編輯 Office 檔案

Dropbox - 雲端同步老牌

雖然 iCloud 很方便,但 Dropbox 跨平台支援更好。Windows、Linux、手機都能無縫同步。

我習慣把重要專案和文件都放 Dropbox,換電腦也不用擔心資料不見。

什麼時候用: 跨裝置同步、團隊共享資料

堅果雲 - 國產雲端同步

如果你常跟中國的團隊協作,堅果雲會更順。它在中國的同步速度比 Dropbox 快很多。

而且支援 WebDAV,可以配合很多筆記 App 用。

什麼時候用: 兩岸協作、需要 WebDAV 同步

開發者工具:寫 Code 必裝 💻

如果你是工程師或想學寫程式,這些工具能讓開發環境專業起來。

iTerm2 - 終端機強化版

Mac 內建的終端機很基本,iTerm2 是加強版。支援分割視窗、搜尋歷史指令、自動完成、快速複製貼上⋯⋯功能多到用不完。

寫程式的人幾乎都用 iTerm2。裝完再搭配 Oh My Zsh,你的終端機會帥到不行。

什麼時候用: 常用終端機的開發者、系統管理

Visual Studio Code - 程式碼編輯器之王

VS Code 根本是現在最多人用的程式碼編輯器。免費、開源、擴充套件超豐富,什麼語言都能寫。

內建 Git 整合、偵錯工具、智慧補全,而且啟動速度快。微軟難得做對一件事。

什麼時候用: 寫程式、改設定檔、處理程式碼

Sublime Text - 輕量編輯器

如果你覺得 VS Code 太肥,Sublime Text 更輕量。開啟速度超快,介面簡潔,適合快速編輯檔案。

我通常用它改設定檔、寫 Markdown、處理大型文字檔。

什麼時候用: 快速編輯文字檔、處理大檔案

Google Antigravity - AI 開發平台新星

這是 Google 在 2025 年 11 月剛推出的 AI 開發工具,目前完全免費。

它基於 VS Code,但整個操作邏輯改成「AI Agent 協作」。你用自然語言描述需求,AI 幫你生成程式碼、除錯、重構。支援 Gemini 3、Claude Sonnet 4.5、GPT 等多種模型。

如果你對 AI 輔助開發有興趣,值得試試。這工具才剛出來,還在快速進化。

什麼時候用: AI 輔助開發、想嘗試新工具

wget - 下載工具

Command Line 下載檔案的經典工具。簡單、可靠、功能強。

下載大檔案、寫自動化腳本的時候會用到。

什麼時候用: 指令列下載、自動化腳本

yt-dlp - 影片下載神器

想下載 YouTube、Vimeo、Twitter 影片？yt-dlp 最強。

指令簡單、支援超多網站、還能選畫質和字幕。學習資源、會議錄影都能輕鬆存下來。

什麼時候用: 下載線上影片、收藏教學資源

aria2 - 多線程下載加速

下載大檔案的時候,aria2 可以同時開多個連線加速,比單線快很多。

支援 HTTP、FTP、BT、磁力連結。進階使用者必備。

什麼時候用: 下載大檔案、需要續傳

ffmpeg - 影音轉檔瑞士刀

處理影片、音訊的萬用工具。轉格式、剪輯、合併、提取音軌⋯⋯只要是影音處理,ffmpeg 都能搞定。

雖然是指令列工具,但功能強到很多圖形軟體底層都用它。

什麼時候用: 批次轉檔、處理影音、自動化處理

創作 + 娛樂:影音、社群、BBS 🎬

工作之餘也要放鬆一下,這些工具讓你的 Mac 更全能。

CapCut - 免費剪片神器

抖音母公司 ByteDance 出品的剪片軟體,國際版叫 CapCut。

重點是完全免費！濾鏡、特效、音樂、貼紙全都能用,不用付錢買會員。介面簡單好上手,剪 YouTube 影片、Vlog、短影音都很適合。

Mac 版本建議裝 5.9 或更早版本,6.x 版本要 TikTok 帳號登入比較麻煩。

什麼時候用: 剪影片、做社群短影音

Firefox - 注重隱私的瀏覽器

雖然 Mac 有內建 Safari,但 Firefox 對隱私保護更好,而且跨平台同步方便。

擴充套件豐富、開發者工具強,很多工程師都用 Firefox。

什麼時候用: 注重隱私、要跨平台同步

KKBOX - 華語音樂首選

台灣本土的音樂串流服務,華語歌曲特別齊全。

如果你主要聽華語音樂,KKBOX 的曲庫比 Spotify 完整很多。

什麼時候用: 聽華語流行音樂

foobar2000 - 發燒友音樂播放器

如果你有收藏高品質音樂檔(FLAC、APE),foobar2000 是專業音樂播放器。

音質好、可自訂性高、支援各種音訊格式。發燒友愛用。

什麼時候用: 播放本地音樂、追求音質

WeChat / Telegram / WhatsApp - 通訊三寶

這三個通訊軟體各有用途:

  • WeChat: 跟中國朋友聯絡、工作溝通必備
  • Telegram: 注重隱私、群組功能強、很多技術社群都用
  • WhatsApp: 國際最多人用、跟國外客戶聯絡方便

根據需求選,或是全裝也行。

什麼時候用: 即時通訊、工作溝通、國際聯繫

Welly - BBS 瀏覽器

對,PTT 還活著,而且活得好好的。

Welly 是 Mac 上最好用的 BBS 瀏覽器,連線穩、功能齊。想上 PTT 逛八卦板、找資訊,裝 Welly 就對了。

什麼時候用: 上 PTT、其他 BBS 站

一鍵安裝魔法:Homebrew 批次腳本 🚀

看到這你可能會想:「這麼多軟體,要一個一個裝到什麼時候？」

別擔心,Mac 有個神器叫 Homebrew,用指令就能快速裝軟體。

Homebrew 是什麼？

把它想成 Mac 的應用程式商店,只是用指令操作。一行指令就能裝好軟體,更新也超方便。

安裝 Homebrew

打開終端機(或 iTerm2),貼上這行指令:

1

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

等個幾分鐘,Homebrew 就裝好了。

#!/bin/bash

echo "🚀 開始安裝 Mac 必備軟體..."

# 檢查是否已安裝 Homebrew
if ! command -v brew &> /dev/null; then
    echo "📦 安裝 Homebrew..."
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
fi

# 更新 Homebrew
echo "🔄 更新 Homebrew..."
brew update

# 安裝指令列工具
echo "⚙️  安裝指令列工具..."
brew install wget
brew install yt-dlp
brew install aria2
brew install ffmpeg

# 安裝系統輔助工具
echo "🛠️  安裝系統輔助工具..."
brew install --cask iterm2
brew install --cask raycast
brew install --cask rectangle
brew install --cask clipy
brew install --cask appcleaner
brew install --cask the-unarchiver
brew install --cask snipaste

# 安裝生產力工具
echo "📝 安裝生產力工具..."
brew install --cask notion
brew install --cask obsidian
brew install --cask anki
brew install --cask libreoffice
brew install --cask dropbox

# 安裝開發工具
echo "💻 安裝開發工具..."
brew install --cask visual-studio-code
brew install --cask sublime-text

# 安裝通訊 & 娛樂軟體
echo "🎬 安裝通訊與娛樂軟體..."
brew install --cask firefox
brew install --cask telegram
brew install --cask wechat
brew install --cask whatsapp
brew install --cask kkbox
brew install --cask foobar2000
brew install --cask welly

echo "✅ 安裝完成！"
echo "💡 部分軟體可能需要手動下載:"
echo "   - PopClip (Mac App Store 付費軟體)"
echo "   - 堅果雲 (官網下載: https://www.jianguoyun.com/)"
echo "   - CapCut (官網下載: https://www.capcut.cn/)"
echo "   - Google Antigravity (官網下載: https://antigravity.google/)"

使用方法:

  1. 複製整段腳本
  2. 貼到終端機
  3. 按 Enter
  4. 去泡杯咖啡,等它跑完

跑完之後,大部分軟體就裝好了。少數幾個(PopClip、堅果雲、CapCut、Google Antigravity)要手動去官網下載,但也很快。

開始探索你的 Mac 🎉

裝完這些軟體,你的 Mac 基本上已經是一台戰鬥力滿點的生產力機器了。

但記住:工具是起點,習慣才是關鍵。

裝了 Raycast 不代表你就會變高效,重點是養成「用快捷鍵啟動」的習慣。裝了 Notion 不代表你就能管好專案,重點是建立自己的工作流程。

我的建議:

  1. 不要一次裝太多,先挑幾個最需要的裝
  2. 花時間習慣新工具,至少用一週再決定要不要留
  3. 慢慢探索,很多工具都有隱藏功能等你發掘

Mac 是個很棒的平台,配上對的工具,它能幫你完成很多事。

開始探索吧,祝你用得開心！💪
