## Github tool: OpenResearch AI生成企業財務報告

這是一個 Github OpenResearch tool 產業研究報告產生器。

https://github.com/dmquant/OpenResearch

可以透過 AI 搜尋產業資料並整合成一份專業的產業、公司研究報告。

### 測試生成研究報告結果

https://goooolai.zeabur.app/1128TW_2368.html

### 使用方式

1. 在 Github 下載 code 後,使用 VS Code 開啟下載下來的檔案
2. 可以使用 Claude / Gemini tool
3. 請 AI coding tool 開啟此專案資料並輸入 Gemini API key
4. 就可以開始執行此專案
