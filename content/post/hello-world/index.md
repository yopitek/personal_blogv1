## 歡迎！

這是使用 Hugo Stack 主題建立的第一篇文章。

### 關於這個部落格

這個部落格使用了以下技術：

- **Hugo** - 快速的靜態網站生成器
- **Stack Theme** - 美觀且功能豐富的 Hugo 主題
- **GitHub Pages** - 免費的網站託管服務

### 特色功能

Stack 主題提供了許多實用功能：

1. 響應式設計
2. 深色模式支援
3. 全文搜尋
4. 文章分類和標籤
5. 目錄導航

### 程式碼範例

這個主題也支援程式碼高亮：

```python
def hello_world():
    print("Hello, World!")
    return "歡迎來到我的部落格"

if __name__ == "__main__":
    hello_world()
```

```javascript
function greet(name) {
    console.log(`Hello, ${name}!`);
}

greet("World");
```
