# Personal Blog

這是使用 Hugo Stack theme 建立的個人部落格。

## 本地開發

```bash
# 啟動開發伺服器
hugo server -D

# 建立新文章
hugo new content/post/文章名稱/index.md
```

## 部署

推送到 GitHub 的 main 分支會自動觸發部署到 GitHub Pages。

## 網站連結

- 本地預覽: http://localhost:1313/personal_blog/
- 線上網站: https://yopitek.github.io/personal_blog/

## 技術棧

- Hugo (靜態網站生成器)
- Hugo Stack Theme
- GitHub Pages (託管)
- GitHub Actions (自動部署)
